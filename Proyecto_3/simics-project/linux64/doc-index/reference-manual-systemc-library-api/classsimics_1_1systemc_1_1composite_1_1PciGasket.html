Detailed Description

template<unsigned int BUSWIDTH = 32, typename TYPES = tlm::tlm_base_protocol_types>
class simics::systemc::composite::PciGasket< BUSWIDTH, TYPES >

Composite Pci Gasket to help the wrapping of SystemC PCI devices in Simics.

Provides an interconnect module that snoops the transactions to config space and handles the Simics specific mapping required to forward transactions on pci-bus io or memory space to the corresponding BAR target socket.
