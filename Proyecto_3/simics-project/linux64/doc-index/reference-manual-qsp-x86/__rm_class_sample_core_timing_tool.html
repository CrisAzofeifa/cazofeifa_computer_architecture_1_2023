sample_core_timing_tool

Provided By
sample-core-timing
Description
Sample x86 performance tool.
Interfaces Implemented
conf_object, instrumentation_tool, log_object

Attributes

background_activity

Pseudo attribute; read/write access; type: float. Background activity per cycle

base_activity

Pseudo attribute; read/write access; type: float. Baseline activity per instruction

base_cycles

Pseudo attribute; read/write access; type: float. Baseline cycles per instruction

connections

Pseudo attribute; read-only access; type: [o*]. Example attribute, returns the connections

instruction_classes

Pseudo attribute; read/write access; type: [[sff]*]. List of instruction classes. Each class consists of a string to look for, fraction of extra cycles per instance and extra activity per instance

read_activity

Pseudo attribute; read/write access; type: float. Extra activity per explicit read access

read_cycles

Pseudo attribute; read/write access; type: float. Extra cycles per explicit read access

write_activity

Pseudo attribute; read/write access; type: float. Extra activity per explicit write access

write_cycles

Pseudo attribute; read/write access; type: float. Extra cycles per explicit write access
