X86_Vmcs_Write

Provided By
x86-intel64
Description
Triggered when Intel® Virtual Machine Control Structure (Intel® VMCS) is written by VMWRITE instruction. The hap is called before the write is performed. When the callback is called, the VMCS field has not yet been updated with the new value.
Callback Type
void (*)(lang_void *callback_data, 
         conf_object_t *trigger_obj, int64 field_index, 
         int64 value);

Index
field_index
