x58-remap-unit

Provided By
X58-devices
Description
The DMA/Interrupt remapping unit
Interfaces Implemented
conf_object, log_object, transaction_translator
Port Objects
bank.vtd (bank_instrumentation_subscribe, instrumentation_order, io_memory, register_view, register_view_read_only)
port.HRESET (signal)
port.interrupt (transaction)

Attributes

apic_bus

Optional attribute; read/write access; type: [os], object, or nil. APIC bus to handle interrupts

Required interfaces: apic_bus.

memory_space

Optional attribute; read/write access; type: [os], object, or nil. Physical Memory

non_isoch

Optional attribute; read/write access; type: boolean. TRUE if this is the Non-Isochronous (i.e. first) VT-d engine on the X58

use_vtd_interrupt_decoding

Optional attribute; read/write access; type: boolean. Use VT-d address decoding to obtain handles when remapping interrupts, rather than x58 decoding

Command List

Commands
infoprint information about the object
statusprint status of the object
