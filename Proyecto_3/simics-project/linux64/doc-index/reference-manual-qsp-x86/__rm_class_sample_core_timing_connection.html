sample_core_timing_connection

Provided By
sample-core-timing
Description
Sample x86 core timing connection
Interfaces Implemented
conf_object, instrumentation_connection, log_object, telemetry

Attributes

cpu

Pseudo attribute; read-only access; type: object. Returns the connected cpu

dynamic_events

Pseudo attribute; read-only access; type: [[si]*]. Statistics for instruction classes.

static_events

Pseudo attribute; read-only access; type: [[si]*]. Statistics for static events, like read, write and total number of instructions.

telemetry_data

Pseudo attribute; read-only access; type: [[sff]*]. Telemetry for the core together with average values.

Command List

Commands

statusprint status of the object
