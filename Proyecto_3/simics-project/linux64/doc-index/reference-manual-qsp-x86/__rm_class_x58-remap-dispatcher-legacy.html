x58-remap-dispatcher-legacy

Provided By
X58-legacy
Aliases
x58_remap_dispatcher
Description
Translate dispatcher unit. - legacy variant, using old PCIe libraries
Interfaces Implemented
conf_object, log_object, pci_upstream

Attributes

default_remapping_unit

Optional attribute; read/write access; type: [os], object, or nil. The default dma remapping unit.

Required interfaces: pci_upstream.

gfx_objs

Optional attribute; read/write access; type: [o*]. List of configuration objects from which transactions will be remapped using the GFX remap unit.

gfx_remapping_unit

Optional attribute; read/write access; type: [os], object, or nil. The dma remapping unit used for gfx.

Required interfaces: pci_upstream.
