QSP Clear Linux Package Release Notes

Introduction

This document identifies important information for the QSP Clear Linux Package package for Simics 6. All users of this specific package should review this document carefully.

Minor Changes Since Simics 6.0.0

The following is a list of changes since the initial Simics 6.0.0 release.

  • 6.0.10 (build 6110)

    • Common
      • Simics no longer comes with documentation in PDF format.

  • 6.0.7 (build 6082)

    • Common
      • Documentation in HTML format has been added.

  • 6.0.6 (build 6050)

    • Common
      • Fixed a problem on Windows where the installer still asked for the Model Builder key (bug #SIMINT-1236).
      • Support for Python 2.7 has been removed. Simics now only embeds a Python 3 interpreter.

  • 6.0.5 (build 6030)

    • Common
      • The separation of source code into separate packages has been removed. Source code is now included directly into each package.

  • 6.0.4 (build 6028)

    • Common
      • The installers for Linux and macOS have been updated. Support for fallback mode of decryption keys has been removed.

  • 6.0.3 (build 6021)

    • QSP Clear Linux
      • The Clear Linux image was updated. The updated image got a few utilities installed like telnet and ping. SimicsFS was reconfigured for users' convenience. E.g., access to the /host directory is allowed for a user account now. SSH daemon on the image was configured to allow login with an empty password as well as login by the "root" user. Also, automatic time synchronization with time servers on the Web was disabled to avoid undesired date changes and unwanted network traffic. The released image has two accounts configured: "root" and "simics". Both accounts have empty passwords. Please use older versions of the package if previously shipped image files are needed (bug #SIMICS-13675).

  • 6.0.2 (build 6015)

    • QSP Clear Linux
      • The Clear Linux used in the package was updated. Please use older versions of the package if previously shipped image files are needed. The released image has two accounts configured: "root" and "simics". Both accounts have empty passwords.

  • 6.0.0 (build 6006)

    • QSP Clear Linux
      • The first release of the package. The cl-lite-b27930.craff image contains a pre-installed Clear Linux distribution. The system was configured for use with the QSP-x86 and x58-ICH10 virtual platforms. Username/password for a regular user account is "simics"/"simics", for a super-user account - "root"/"root".
