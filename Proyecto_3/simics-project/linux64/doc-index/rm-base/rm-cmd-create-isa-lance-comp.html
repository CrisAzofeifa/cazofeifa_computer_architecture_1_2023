create-isa-lance-comp

Synopsis

create-isa-lance-comp ["name"] [base_port] [irq_level] "mac_address"

Description

This command creates a non-instantiated component of the class isa_lance_comp.

The class description for the isa_lance_comp class: The "isa_lance_comp" component represents an ISA bus based Ethernet adapter.

name is Optional
If not specified, the component will get a class-specific default name.

base_port is Optional
The starting port number in I/O space. The default port is 0x300, and the mapping is 0x17 bytes large.

irq_level is Optional
The interrupt level for the Lance device, default is 7.

mac_address is Required
The MAC address of the Ethernet adapter.

Provided By

isa-comp
