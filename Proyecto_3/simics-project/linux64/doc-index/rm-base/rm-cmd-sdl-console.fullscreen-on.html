<sdl-console>.fullscreen-on

Synopsis

<sdl-console>.fullscreen-on

Description

Enable fullscreen mode. The console will try to reopen itself in fullscreen mode the next time input is grabbed. When input grabbing is later disabled, the console will go back to a normal window.

Provided By

sdl-console
