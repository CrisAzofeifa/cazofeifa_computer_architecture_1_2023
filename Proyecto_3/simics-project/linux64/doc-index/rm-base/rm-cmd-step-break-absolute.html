step-break-absolute – deprecated

Synopsis

step-break-absolute [cpu-name] instructions
<step>.step-break-absolute instructions

Description

Sets a breakpoint that will trigger when the selected processor's step counter has reached the instructions value. If no processor is specified in cpu-name, the currently selected frontend processor is used.

Provided By

Simics Core
