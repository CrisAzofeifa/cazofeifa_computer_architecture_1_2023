x86_mode_filter

Description

Objects of this class are used to filter an instrumentation tool's data collection according to x86 execution modes.

Interfaces Implemented

conf_object, log_object, instrumentation_filter_master

Commands for this class

  • add-mode – add a mode to the filter
  • delete – add a mode to the filter
  • info – print information about the object
  • remove-mode – remove a mode from the filter
  • status – print status of the object

Attributes

modes
