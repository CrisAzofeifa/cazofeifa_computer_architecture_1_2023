create-pci-i82543gc

Synopsis

create-pci-i82543gc ["name"] "mac_address" ["bios"]

Description

Creates a non-instantiated component of the class "pci-i82543gc". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

mac_address - The MAC address of the Ethernet adapter. bios - The x86 BIOS file to use.
