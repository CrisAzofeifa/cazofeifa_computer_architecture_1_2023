<sc_memory_profiler_control>.memory-profiler-results

Synopsis

<sc_memory_profiler_control>.memory-profiler-results [limit] [object]

Description

Lists memory usage of method and thread processes, as well as the number of allocations. limit specifies how many processes are listed in the result. object limits the scope of listed processes to only those under its hierarchy.
