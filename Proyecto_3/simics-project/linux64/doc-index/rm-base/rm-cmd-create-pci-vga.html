create-pci-vga

Synopsis

create-pci-vga ["name"] ["bios"]

Description

Creates a non-instantiated component of the class "pci-vga". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

bios - The VGA BIOS file to use.
