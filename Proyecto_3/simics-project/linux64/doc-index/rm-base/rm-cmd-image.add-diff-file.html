<image>.add-diff-file

Synopsis

<image>.add-diff-file filename [-replace] [-rw] [-force]

Description

Adds a diff file, filename, to the list of files for an image. The diff file was typically created with save-diff-file, or by a saved configuration.

The file can be treated as writable instead of read-only using the -rw flag. The clear-files command should be used instead of the deprecated -replace flag.

If any unsaved changes are found in the image object, the command will fail. Such changes can be discarded by using the -force flag, or first saved using the save-diff-file command.
