<component>.disconnect – deprecated

Synopsis

<component>.disconnect ["connector"] [component] ["dst-connector"] [-quiet]

Description

Disconnect the connector from another component connector, identified by component dst-connector. Connectors can only be disconnected if they support hotplugging. The -quiet flag inhibits all output.
