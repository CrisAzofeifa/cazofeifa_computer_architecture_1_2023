<sc_memory_profiler_control>.enable-memory-profiler

Synopsis

<sc_memory_profiler_control>.enable-memory-profiler

Description

Enables memory profiling for SystemC processes. This allows inspection of the number and size of allocations performed in a SystemC thread or method.
