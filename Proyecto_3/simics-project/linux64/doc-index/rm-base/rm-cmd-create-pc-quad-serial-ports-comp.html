create-pc-quad-serial-ports-comp

Synopsis

create-pc-quad-serial-ports-comp ["name"]

Description

This command creates a non-instantiated component of the class pc_quad_serial_ports_comp.

The class description for the pc_quad_serial_ports_comp class: The "pc_quad_serial_ports_comp" component represents four PC compatible serial ports.

name is Optional
If not specified, the component will get a class-specific default name.
