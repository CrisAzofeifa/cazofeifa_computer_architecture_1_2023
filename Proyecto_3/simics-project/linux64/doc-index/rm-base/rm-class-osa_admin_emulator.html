osa_admin_emulator

Description

This class forwards calls to the interfaces previously implemented by the 'osa_admin' class to the parent object.

Interfaces Implemented

conf_object, log_object, osa_control, osa_control_v2, osa_machine_notification, osa_machine_query, osa_node_path, osa_node_tree_admin, osa_node_tree_notification, osa_node_tree_query, osa_tracker_state_admin, osa_tracker_state_notification, osa_tracker_state_query

Commands for this class

  • info – print information about the object
  • status – print status of the object

os-awareness
