<process_histogram>.add-instrumentation

Synopsis

<process_histogram>.add-instrumentation software ["group"]

Description

Connects the tool to one software as given by the software argument.

The optional argument group let a user specify a named instrumentation group to use for the connection. (See add-instrumentation-group for details on named groups.)
