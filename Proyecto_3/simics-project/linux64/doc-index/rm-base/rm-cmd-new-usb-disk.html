new-usb-disk

Synopsis

new-usb-disk ["name"] [size] ["file"]

Description

Creates an instantiated component of the class "usb-disk". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

size - The size of the disk in bytes. file - File with disk contents for the full disk, either a raw file or a CRAFF file.

Provided By

usb-components
