switch-mouse-to-tablet

Synopsis

switch-mouse-to-tablet [-disable] [-dry-run] [-enable]

Description

The switch-mouse-to-tablet command creates and connects a usb tablet for each graphical console (gfx_console_comp) that it finds, plus switches the mouse input from relative to absolute positioning. The -dry-run flag will only list the affected consoles without actually doing anything. The -enable and -disable flags will start or stop new graphical consoles from being automatically updated with an absolute tablet pointer.
