new-server-console-comp

Synopsis

new-server-console-comp ["name"] [telnet_port]

Description

This command creates an instantiated component of the class server_console_comp.

The class description for the server_console_comp class: Deprecated class - do not use

name is Optional
If not specified, the component will get a class-specific default name.

telnet_port is Unknown
TCP/IP port to connect the telnet service of the console to.
