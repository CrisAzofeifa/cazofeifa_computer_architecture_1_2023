create-sio-w83627hf-comp

Synopsis

create-sio-w83627hf-comp ["name"] [add_par_port]

Description

This command creates a non-instantiated component of the class sio_w83627hf_comp.

The class description for the sio_w83627hf_comp class: Winbond W83627HF I/O.

name is Optional
If not specified, the component will get a class-specific default name.

add_par_port is Optional
Set to TRUE to add a parallel port to the Super I/O device. Default is FALSE since the current implementation is a dummy device.

Provided By

isa-comp
