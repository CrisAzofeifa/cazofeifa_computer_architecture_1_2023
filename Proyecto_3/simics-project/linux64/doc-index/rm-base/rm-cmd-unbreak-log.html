unbreak-log – deprecated

Synopsis

unbreak-log [object]

Description

Disables breaking simulation on logs. Without argument, all breaking on logs is disabled.

The object argument specifies which object to disable breaking for.

Provided By

Simics Core

See Also

bp.delete, <bp-manager.log>.break
