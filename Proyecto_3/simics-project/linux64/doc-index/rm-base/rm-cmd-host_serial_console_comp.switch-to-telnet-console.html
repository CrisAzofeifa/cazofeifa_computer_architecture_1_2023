<host_serial_console_comp>.switch-to-telnet-console

Synopsis

<host_serial_console_comp>.switch-to-telnet-console [port]

Description

Replace the host_serial_console_comp component with a telnet_console_comp component. The argument port is the port number thata telnet client can connect to. If no argument is given, a free port on the host is used. This port number is returned by the command.

Provided By

std-comp
