create-phy-mii-transceiver

Synopsis

create-phy-mii-transceiver ["name"] [phy_id] [mii_address]

Description

Creates a non-instantiated component of the class "phy-mii-transceiver". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

phy_id - PHY ID (i.e. vendor) mii_address - PHY address on MII bus
