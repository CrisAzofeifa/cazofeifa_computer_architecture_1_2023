insert-ieee-802-15-4-probe

Synopsis

insert-ieee-802-15-4-probe device ["name"]

Description

Insert IEEE 802.15.4 probe to listen to traffic on the given device, an ieee_802_15_4_receiver.

The probe may be given a name.
