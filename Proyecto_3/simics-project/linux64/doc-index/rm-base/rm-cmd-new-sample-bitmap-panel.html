new-sample-bitmap-panel

Synopsis

new-sample-bitmap-panel ["name"]

Description

This command creates an instantiated component of the class sample_bitmap_panel.

The class description for the sample_bitmap_panel class: Simple panel demo, with a bitmap led and a bitmap button in a canvas with a background image.

name is Optional
If not specified, the component will get a class-specific default name.
