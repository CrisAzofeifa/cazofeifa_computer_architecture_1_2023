<breakpoint_manager>.list – deprecated

Synopsis

<breakpoint_manager>.list

Description

Lists all breakpoints known by the breakpoint manager. For each breakpoint you get its id, and a short summary.

To get more details about a breakpoint use the show command.

old-breakpoint-manager
