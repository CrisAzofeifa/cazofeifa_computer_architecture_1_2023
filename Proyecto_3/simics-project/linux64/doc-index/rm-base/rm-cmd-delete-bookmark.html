delete-bookmark

Synopsis

delete-bookmark ("bookmark"|-all)

Description

Delete the time bookmark specified by the bookmark argument. Or use the -all flag to delete all time bookmarks which can improve forward simulation performance.

Reverse operations are possible in the region following the first (i.e. oldest) time bookmark.

Provided By

Simics Core
