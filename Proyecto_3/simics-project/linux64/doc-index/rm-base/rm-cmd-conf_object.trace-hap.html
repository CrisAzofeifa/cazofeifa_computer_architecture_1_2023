<conf_object>.trace-hap – deprecated

Synopsis

<conf_object>.trace-hap ("hap"|-all|-list)
<conf_object>.untrace-hap ("hap"|-all|-list)

Description

Enables and disables tracing of haps for specific objects. When this is enabled, a message is printed every time the specified hap in specified object is triggered.

The hap parameter specifies the hap.

Instead of a hap, the -all flag may be given. This will enable or disable tracing of all haps.

Using the -list argument will print out the haps currently being traced.

Provided By

Simics Core
