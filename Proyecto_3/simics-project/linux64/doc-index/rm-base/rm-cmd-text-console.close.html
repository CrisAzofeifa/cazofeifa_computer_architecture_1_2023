<text-console>.close

Synopsis

<text-console>.close

Description

Close the console window. When the console window is closed, any output is redirected to the main Simics window. This can be suppressed with the enable-quiet command. To provide input to the target machine while the console is closed, use the input command.

Provided By

xterm-console
