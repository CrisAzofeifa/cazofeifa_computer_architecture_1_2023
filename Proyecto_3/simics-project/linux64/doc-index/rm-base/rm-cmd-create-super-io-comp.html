create-super-io-comp

Synopsis

create-super-io-comp ["name"] [add_par_port]

Description

This command creates a non-instantiated component of the class super_io_comp.

The class description for the super_io_comp class: The "super_io_comp" component represents a generic Super I/O device with legacy PC devices such as two serial ports, one PS/2 keyboard and mouse controller, one floppy device and a parallel port.

name is Optional
If not specified, the component will get a class-specific default name.

add_par_port is Optional
Set to TRUE to add a parallel port to the Super I/O device. Default is FALSE since the current implementation is a dummy device.
