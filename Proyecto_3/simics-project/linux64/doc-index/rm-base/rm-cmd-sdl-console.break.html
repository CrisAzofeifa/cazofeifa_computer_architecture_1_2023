<sdl-console>.break

Synopsis

<sdl-console>.break filename

Description

Activates a previously saved breakpoint specified by filename and returns a breakpoint id. When a graphical breakpoint is reached it is immediately deleted and a hap with id Gfx_Break_String is triggered. If no callbacks for this hap were registered the simulation is stopped.
