step-counter-extension

Description

The step-counter-extension extension class is used to augment objects with a step interface.

Interfaces Implemented

step

Port Objects

vtime.steps
fixed-clock – step queue

Commands for interface step

  • bp-break-step – set step breakpoint
  • bp-run-until-step – run until specified step
  • bp-trace-step – enable tracing of step points
  • bp-wait-for-step – wait for specified step
  • force-step-instruction – step instructions while other processors stall
  • step-break – deprecated set time breakpoint
  • step-break-absolute – deprecated set step breakpoints
  • wait-for-step – deprecated wait until reaching step
