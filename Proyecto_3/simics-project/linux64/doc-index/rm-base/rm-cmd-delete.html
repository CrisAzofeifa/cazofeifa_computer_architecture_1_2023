delete – deprecated

Synopsis

delete ([ id ... ] | list of id) [-all]

Description

Removes the breakpoints matching the ID numbers id.

Use list-breakpoints to list all breakpoints' ID.

If -all is given, delete all breakpoints.

Provided By

Simics Core

See Also

bp.delete, <breakpoint>.bp-break-memory, <bp_manager>.delete, <bp_manager>.enabled, <bp_manager>.ignore-count, <bp_manager>.list
