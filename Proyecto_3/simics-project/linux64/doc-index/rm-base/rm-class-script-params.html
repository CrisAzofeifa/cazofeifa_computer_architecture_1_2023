script-params

Description

Singleton that contains parameters of the current script

Interfaces Implemented

conf_object, log_object

Commands for this class

  • assigned – return names of assigned parameters
  • describe – display parameter information
  • dump – dump parameter values
  • get – get parameter value
  • help – display script parameters
  • info – print information about the object
  • list – list parameter values
  • save – save parameter values
  • setdefault – set parameter value default
  • status – print status of the object

Attributes

current
Pseudo attribute; read-only access; type: [s*]

Simics Core
