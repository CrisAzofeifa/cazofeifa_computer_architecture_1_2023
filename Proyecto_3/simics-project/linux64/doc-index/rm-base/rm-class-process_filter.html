process_filter

Description

Instrumentation filter for processes.

Interfaces Implemented

conf_object, log_object, instrumentation_filter_master

Commands for this class

  • add-pattern – filter on a specific node path pattern
  • delete – remove the filter
  • info – print information about the object
  • remove-pattern – stop filtering on a process
  • status – print status of the object

Attributes

software_comp
Required attribute; read/write access; type: o
OSA software component to use.
tracked_processes
Session attribute; read/write access; type: [s*]
The processes being tracked.
