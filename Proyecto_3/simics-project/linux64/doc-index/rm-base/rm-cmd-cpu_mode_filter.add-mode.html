<cpu_mode_filter>.add-mode

Synopsis

<cpu_mode_filter>.add-mode mode

Description

Restrict instrumentation to only a specific mode. The mode argument can be one of: user, supervisor, or hypervisor.

Provided By

cpu-mode-filter
