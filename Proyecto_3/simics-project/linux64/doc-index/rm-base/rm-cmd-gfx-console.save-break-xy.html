<gfx-console>.save-break-xy

Synopsis

<gfx-console>.save-break-xy filename left top right bottom ["comment"]

Description

Lets the user select a rectangular area inside the graphics console by specifying the top left corner (left, top) and the bottom right corner (right, bottom). The selected area will be saved as a binary graphical breakpoint to the file filename. An optional comment may be added that will be put in the beginning of the file.
