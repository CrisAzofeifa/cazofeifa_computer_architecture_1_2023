create-pc-floppy-controller-comp

Synopsis

create-pc-floppy-controller-comp ["name"]

Description

This command creates a non-instantiated component of the class pc_floppy_controller_comp.

The class description for the pc_floppy_controller_comp class: The "pc_floppy_controller_comp" component represents a legacy pc floppy controller with two attached drives.

name is Optional
If not specified, the component will get a class-specific default name.
