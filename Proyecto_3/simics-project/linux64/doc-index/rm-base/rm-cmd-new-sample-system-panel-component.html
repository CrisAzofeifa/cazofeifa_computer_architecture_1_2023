new-sample-system-panel-component

Synopsis

new-sample-system-panel-component ["name"]

Description

This command creates an instantiated component of the class sample_system_panel_component.

The class description for the sample_system_panel_component class: A basic component to enable debug/demo of various System Panels.

name is Optional
If not specified, the component will get a class-specific default name.
