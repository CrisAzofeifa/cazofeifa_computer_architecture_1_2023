connect-components – deprecated

Synopsis

connect-components src-component ["src-connector"] [component] ["dst-connector"] [-f]

Description

Create a connection between the connectors src-connector and dst-connector of the src-component and component components. Indexed connectors must be quoted. The -f flag selects the first unused connector (in alphabetic order).
