tcf-context-proxy

Description

TCF Context Proxy

Interfaces Implemented

conf_object, log_object, symdebug, run_control

Commands for this class

  • bp-break-source-line – add breakpoint at a source code line
  • bp-break-source-location – add breakpoint at a location
  • bp-run-until-source-line – run until reaching source code line
  • bp-run-until-source-location – run until reaching location
  • bp-trace-source-line – enable tracing of source code line reaches
  • bp-trace-source-location – enable tracing of location reaches
  • bp-wait-for-source-line – wait until reaching source code line
  • bp-wait-for-source-location – wait until reaching location
  • break-line – deprecated add breakpoint at a source code line
  • break-location – deprecated add breakpoint at a location
  • info – print information about the object
  • run-until-line – deprecated run until reaching source code line
  • run-until-location – deprecated run until reaching location
  • status – print status of the object
  • wait-for-line – deprecated wait in the script branch until reaching source code line
  • wait-for-location – deprecated wait in the script branch until reaching location

Commands for interface symdebug

  • down – deprecated go down N stack frames
  • finish-function – deprecated finish the current function
  • frame – deprecated change current stack frame
  • list – deprecated list source and/or disassemble
  • next-instruction – deprecated run to the next instruction, skipping subroutine calls
  • next-line – deprecated run to the next source line, skipping subroutine calls
  • reverse-next-instruction – deprecated reverse to the previous instruction, skipping subroutine calls
  • reverse-next-line – deprecated reverse to the previous source line, skipping subroutine calls
  • reverse-step-instruction – deprecated reverse to the previous instruction
  • reverse-step-line – deprecated reverse to the previous source line
  • stack-trace – deprecated display stack trace
  • step-instruction – deprecated run to the next instruction
  • step-line – deprecated run to the next source line
  • sym-address – deprecated return the address of expression or source line
  • sym-file – deprecated return source file for function or address
  • sym-function – deprecated return function at a given address
  • sym-line – deprecated return source line for function or address
  • sym-list – deprecated list symbols
  • sym-source – deprecated print source location for function or address
  • sym-string – deprecated evaluate symbolic expression
  • sym-type – deprecated return the type of a symbolic expression
  • sym-value – deprecated evaluate symbolic expression
  • sym-write – deprecated write value to variable
  • uncall-function – deprecated reverse to when the current function was called
  • up – deprecated go up N stack frames

Attributes

agent
Required attribute; read/write access; type: o
The TCF agent owning the TCF context this object is a proxy for
cid
Required attribute; read/write access; type: s
The TCF context ID for the context this object is a proxy for
desc
Required attribute; read/write access; type: [so]|[soi]
Description of what this object is a proxy for. Either ["simobj", object] or ["sw-node", software object, node ID].
