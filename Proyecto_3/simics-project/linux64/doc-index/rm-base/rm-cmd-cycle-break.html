cycle-break – deprecated

Synopsis

cycle-break [cpu-name] cycles
<cycle>.cycle-break cycles

Description

Sets a breakpoint that will trigger when the selected processor's cycle counter has reached cycles number of cycles from the time the command was issued. If no processor is specified in cpu-name, the currently selected frontend processor is used.

Provided By

Simics Core
