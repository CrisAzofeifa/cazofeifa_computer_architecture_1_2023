new-tracer – deprecated

Synopsis

new-tracer [-all] [cell]

Description

Create a new tracer that connects to each CPU's memory space and traces instruction and data accesses. Only processors in the current, or specified, cell are traced unless the -all flag is used. If all processors in the system are traced, multi-threading will be disabled.

Provided By

trace

See Also

new-tracer-tool, <base-trace-mem-hier>.start
