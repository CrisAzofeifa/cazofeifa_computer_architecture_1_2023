create-isa-vga-comp

Synopsis

create-isa-vga-comp ["name"] ["bios"]

Description

This command creates a non-instantiated component of the class isa_vga_comp.

The class description for the isa_vga_comp class: The "isa_vga_comp" component represents an ISA bus based VGA compatible graphics adapter.

name is Optional
If not specified, the component will get a class-specific default name.

bios is Optional
The VGA BIOS file to use.

Provided By

isa-comp
