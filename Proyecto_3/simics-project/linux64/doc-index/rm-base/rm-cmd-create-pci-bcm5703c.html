create-pci-bcm5703c

Synopsis

create-pci-bcm5703c ["name"] "mac_address" ["bios"]

Description

Creates a non-instantiated component of the class "pci-bcm5703c". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

mac_address - The MAC address of the Ethernet adapter. bios - The x86 BIOS file to use.
