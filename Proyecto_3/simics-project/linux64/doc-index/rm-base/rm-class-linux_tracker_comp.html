linux_tracker_comp

Description

A Linux Tracker top object that can be inserted into a domain.

Interfaces Implemented

conf_object, log_object, osa_parameters, osa_tracker_component

Commands for this class

  • detect-parameters – try to detect settings for the Linux tracker
  • info – print information about the object
  • list-kernel-modules – list kernel modules
  • log-syscalls – start logging system calls
  • set-kernel-modules-path – set base path for kernel modules
  • status – print status of the object

Commands for interface osa_parameters

  • load-parameters – load tracker parameter settings
  • save-parameters – save parameters
  • supports-parameters – check if supported
