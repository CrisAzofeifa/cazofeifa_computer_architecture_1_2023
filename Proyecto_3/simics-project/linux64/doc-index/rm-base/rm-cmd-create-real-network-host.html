create-real-network-host

Synopsis

create-real-network-host ["name"] "interface"

Description

Creates a non-instantiated component of the class "real-network-host". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

interface - Interface to connect to

Provided By

real-network
