<sc_process_profiler_control>.process-profiler-results

Synopsis

<sc_process_profiler_control>.process-profiler-results [limit] [object]

Description

Lists information of the total number of calls of method and thread processes, as well as the aggregated executiontime of those. limit specifies how many processes are listed in the result. object limits the scope of listed processes to only those under its hierarchy.
