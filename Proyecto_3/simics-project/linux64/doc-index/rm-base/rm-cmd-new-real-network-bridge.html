new-real-network-bridge

Synopsis

new-real-network-bridge ["name"] ["interface"]

Description

Creates an instantiated component of the class "real-network-bridge". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

interface - Interface to connect to

Provided By

real-network
