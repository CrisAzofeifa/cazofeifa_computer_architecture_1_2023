create-ps2-keyboard-mouse-comp

Synopsis

create-ps2-keyboard-mouse-comp ["name"]

Description

This command creates a non-instantiated component of the class ps2_keyboard_mouse_comp.

The class description for the ps2_keyboard_mouse_comp class: The "ps2_keyboard_mouse_comp" component represents the PS/2 8042 keyboard controller with a connected 105 key keyboard and three button mouse.

name is Optional
If not specified, the component will get a class-specific default name.
