print-blueprint-state

Synopsis

print-blueprint-state ["iface"] ["field"] ["pat"] [-b] [-i]

Description

Inspect blueprint state. If iface is specified, limit output to blueprint interconnects of the specified type. The field argument can be used to restrict the output to just include the specified field.

As an alternative/complement to iface, it is possible to specify pat which selects all interconnects that match the specified substring. This is for instance useful to select interconnects defined in a specific node.

If the -b flag is specified, then all interconnects bound to a specific namespace node are listed. Conversely, the -i flag lists all interconnects subscription points.
