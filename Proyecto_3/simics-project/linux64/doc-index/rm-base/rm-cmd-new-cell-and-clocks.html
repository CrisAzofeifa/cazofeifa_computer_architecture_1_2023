new-cell-and-clocks

Synopsis

new-cell-and-clocks ["name"] [clock_number] [freq_mhz] [domain]

Description

Creates an instantiated component of the class "cell-and-clocks". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

clock_number - Number of clocks to run freq_mhz - Clocks' frequency

Provided By

clock
