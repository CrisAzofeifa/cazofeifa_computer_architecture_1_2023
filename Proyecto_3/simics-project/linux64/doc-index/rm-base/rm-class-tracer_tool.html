tracer_tool

Description

The tracer tool prints every executed instruction, all data accesses, and also all the exceptions that occurs in the connected processors. The data that is printed can be controlled by various arguments to the new-tracer-tool. The output can also be written to a file.

Interfaces Implemented

conf_object, log_object, instrumentation_tool

Commands for this class

  • add-filter – add filter to the tool
  • add-instrumentation – add instrumentation
  • delete – deletes instrumentation tool
  • disable-instrumentation – disable instrumentation
  • enable-instrumentation – enable instrumentation
  • info – print information about the object
  • list-providers – list-provider objects
  • remove-filter – remove filter from the tool
  • remove-instrumentation – remove instrumentation
  • status – print status of the object

Attributes

file
Optional attribute; read/write access; type: s|n
Output file

Provided By

instrumentation-tracer-tool
