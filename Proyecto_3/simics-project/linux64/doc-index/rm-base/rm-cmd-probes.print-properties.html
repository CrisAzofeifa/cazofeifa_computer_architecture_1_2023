<probes>.print-properties

Synopsis

<probes>.print-properties "probe" [-show-key-values]

Description

Print the properties of a probe.

With -show-key-values the key value pairs will be printed as they where implemented by the probe_interface or the probe_index interface method properties.

Without those flags the probe value will be printed without any particular formatting.
