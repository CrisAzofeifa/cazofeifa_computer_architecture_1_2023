simicsfs_server

Description

The SimicsFS server allows a SimicsFS client in the target system to mount all or parts of the host file system.

Interfaces Implemented

conf_object, log_object

Commands for this class

  • add-client-group – add client group
  • add-path-rule – add path rule
  • check-path-rules – check path rules
  • info – print information about the object
  • list-client-groups – list client groups
  • list-clients – list SimicsFS clients
  • list-path-rules – list path rules
  • remove-client-group – remove client group
  • remove-path-rule – remove path rule
  • status – print status of the object
