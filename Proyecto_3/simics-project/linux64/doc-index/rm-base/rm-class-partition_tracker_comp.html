partition_tracker_comp

Description

A Partition Tracker that can be used to assign only certain processors to other trackers.

Interfaces Implemented

conf_object, log_object, osa_parameters, osa_tracker_component

Commands for this class

  • info – print information about the object
  • insert-empty-partition – insert a partition that is not associated with any tracker
  • insert-tracker – insert a new child tracker
  • remove-partition – remove a partition
  • status – print status of the object

Commands for interface osa_parameters

  • load-parameters – load tracker parameter settings
  • save-parameters – save parameters
  • supports-parameters – check if supported

Attributes

guests
Optional attribute; read/write access; type: [[iso|n[o*]]*]
Maps from partitions to guest trackers. Format is [id, name, guest composition, [cpus]]
