list-object-references

Synopsis

list-object-references object [namespace] [-include-ports] [-list-all] [-only-list-objects]

Description

List attributes that references object.

If the -include-ports flag is used attributes referencing port objects belonging to object is included in the list.

By default every non pseudo attribute in every object in the system is searched. With the namespace argument the scope of the search can be reduced.

The -list-all flag lists all references. By default only the first ten references are listed.

By default object, attribute and referenced object is listed. With the -only-list-objects flag only the objects containing an attribute referencing object is listed.

If the command is used in an expression, the command returns a list of lists including referencing object, attribute and object referenced. If the -only-list-objects flag is used only referencing object is included in the list.

Note that the completion time can be very long when run on large platforms.
