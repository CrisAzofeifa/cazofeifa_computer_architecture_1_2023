new-systemc-filter

Synopsis

new-systemc-filter "kind" ["name"]

Description

Creates a new filter with the name given by the name argument and filter the instrumentation connections based on the required kind argument. The kind argument corresponds to a pre-defined set of SystemC objects, currently: signal, event, process, socket.

Provided By

systemc-tools

See Also
