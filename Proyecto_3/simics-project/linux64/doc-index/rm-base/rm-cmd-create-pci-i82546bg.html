create-pci-i82546bg

Synopsis

create-pci-i82546bg ["name"] "mac_address" ["bios"]

Description

Creates a non-instantiated component of the class "pci-i82546bg". If name is not specified, the component will get a class-specific default name.

Component specific arguments:

mac_address - The MAC address of the first Ethernet adapter. The last bit is toggled to give the address for the second interface. mac_address2 - The MAC address of the second Ethernet adapter. It is deduced from the address of the first interface by toggling the last bit. bios - The x86 BIOS file to use.
