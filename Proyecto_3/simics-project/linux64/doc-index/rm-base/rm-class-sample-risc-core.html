sample-risc-core

Description

Sample RISC core.

Interfaces Implemented

conf_object, log_object, context_handler, processor_info_v2, processor_info, int_register, exception, processor_cli, opcode_info

Commands for this class

  • info – print information about the object
  • status – print status of the object

Commands for interface context_handler

  • set-context – set the current context of a CPU

Commands for interface processor_info

  • debug – get debug object
  • disable – switch processor off
  • disassemble – disassemble instructions
  • enable – switch processor on
  • enabled – get enable status
  • load-binary – load binary (executable) file into memory
  • logical-to-physical – translate logical address to physical
  • print-processor-registers – print cpu registers
  • read-reg – read a register
  • set-pc – set the program counter
  • write-reg – write to register
  • x – examine raw memory contents

Commands for interface exception

  • bp-break-exception – set break on exception
  • bp-run-until-exception – run until specified exception occurs
  • bp-trace-exception – enable tracing of exceptions
  • bp-wait-for-exception – wait for specified exception
  • break-exception – deprecated break on CPU exceptions
  • list-exceptions – list exceptions on a processor
  • trace-exception – deprecated trace exceptions
  • unbreak-exception – deprecated break on CPU exceptions
  • untrace-exception – deprecated trace exceptions
  • wait-for-exception – deprecated wait until a processor exception happens
