whitelist-old-component

Synopsis

whitelist-old-component "component-class"

Description

Adds the component class component-class to the deprecation whitelist, preventing deprecation warnings when objects of the class are instantiated. This command can be used to avoid deprecation warnings for old style, non-hierarchical, components while converting configurations to use the new component framework. This command will be removed as the old component system discarded.
