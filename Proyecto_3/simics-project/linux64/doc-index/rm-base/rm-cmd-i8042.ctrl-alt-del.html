<i8042>.ctrl-alt-del

Synopsis

<i8042>.ctrl-alt-del

Description

Sends a Ctrl-Alt-Del command to the console. This is useful primarily on a Windows-hosted Simics to avoid having Ctrl-Alt-Del caught by the host OS.
