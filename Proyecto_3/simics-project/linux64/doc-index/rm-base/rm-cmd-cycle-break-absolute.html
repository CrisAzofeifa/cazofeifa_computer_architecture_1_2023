cycle-break-absolute – deprecated

Synopsis

cycle-break-absolute [cpu-name] cycles
<cycle>.cycle-break-absolute cycles

Description

Sets a breakpoint that will trigger when the selected processor's cycle counter has reached the cycles value. If no processor is specified in cpu-name, the currently selected frontend processor is used.

Provided By

Simics Core
