bp-manager

Description

The set of breakpoints in Simics

Interfaces Implemented

conf_object, log_object, breakpoint_type, breakpoint_registration, bp_manager

bp-manager.step_event – step event breakpoints

bp-manager.time – virtual time breakpoints

Commands for this class

  • info – print information about the object
  • status – print status of the object

Commands for interface bp_manager

  • delete – delete a breakpoint
  • disable – disable breakpoint
  • enable – enable breakpoint
  • enabled – enable or disable breakpoint
  • ignore-count – set number of hits of breakpoint to ignore
  • list – list breakpoints
  • list-types – list breakpoint type providers
  • show – show details about a breakpoint
  • test-trigger – test trigger breakpoint
  • wait-for-breakpoint – wait for breakpoint
