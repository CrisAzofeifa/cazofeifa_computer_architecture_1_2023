<graphcon>.save-break-xy

Synopsis

<graphcon>.save-break-xy filename left top right bottom

Description

Stores a rectangular area of the screen to the file specified by filename, for use with the break-gfx command. The rectangle is specified by the top left corner (left, top) and the bottom right corner (right, bottom), which must define a non-empty rectangle contained in the console screen.

Provided By

graphcon
