<breakpoint_manager>.temporary – deprecated

Synopsis

<breakpoint_manager>.temporary id [value]

Description

Should breakpoint id be deleted after it has triggered the first time. Set status with value set to "yes" or "no". Omit the flag to get current status.

The command returns what the, possibly new, temporary state of the breakpoint is.
