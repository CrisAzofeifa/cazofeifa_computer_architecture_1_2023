pci_downstream

Description
The pci_downstream interface is implemented by PCI(e) buses to handle downstream transactions. 
SIM_INTERFACE(pci_downstream) {
        exception_type_t (*operation)(conf_object_t *obj,
                                      generic_transaction_t *mem_op,
                                      addr_space_t space);
};
